I AM EXTERNAL CSS
The styling of this particular division is defined using external css
External css is defined in a seperate css file which is then linked to the html page using a link tag
I AM INTERNAL CSS
The styling of this particular division is defined using internal css
Internal css is defines inside the head section of the html file using a tag called the style tag
I AM INLINE CSS
The styling of this particular devision is defined using inline css
Inline css is defined inside the particular tag where it is to be used inside the html file
